# homebrew-packguard

Homebrew tap for [PackGuard](https://github.com/Tmauc/packguard).

## Install

```bash
brew tap Tmauc/packguard
brew install packguard
```

Then verify:

```bash
packguard --version
packguard --help
```

## Updating

New PackGuard versions publish signed release artifacts at
<https://github.com/Tmauc/packguard/releases>. The formula in this tap
is auto-regenerated by PackGuard's release workflow on each `v*` tag —
do not hand-edit it.
